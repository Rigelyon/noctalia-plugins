# Wallcards Plugin for Noctalia

A lively wallpaper selector for images and video.

https://github.com/user-attachments/assets/9ffbc83d-95e5-4dcd-a834-7bd224211b55

## Features

- **Image & Video Support**
- **Live Preview**

## Installation

This plugin is part of the `noctalia-plugins` repository. Following package need to be installed: 

```sh
pacman -S magick ffmpeg mpvpaper
```
